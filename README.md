# IRCTC Sprint — Problem Discovery & Design Sprint

A civic tech audit of IRCTC (irctc.co.in) — the platform used by 8 crore Indians every day.

## Part A — Problem Discovery
Six real, evidence-based UX and infrastructure failures documented from live platform exploration.

## Part B — Feature Specifications *(coming soon)*
Each of the 6 problems becomes a detailed feature spec, AI feature proposal, and prioritisation matrix.
